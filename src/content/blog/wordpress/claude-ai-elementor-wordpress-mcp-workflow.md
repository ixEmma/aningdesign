## Step-by-Step Claude AI and Elementor Workflow

### Step 1: Preview the Final Landing Page

Timestamp: 00:01

The tutorial starts by showing the final high-converting landing page. This is important because it gives the workflow a clear target before diving into settings and tools.

When using AI for Elementor design, you should always know what kind of page you are trying to build. Is it a service landing page, a portfolio page, a lead-generation page, or a sales page? The final goal affects the layout, section order, CTA placement, and content hierarchy.

### Step 2: Walk Through CLAUDE.md and Skills

Timestamp: 00:01:55

The next part explains the role of `CLAUDE.md` files and custom skills. A `CLAUDE.md` file works like project documentation for Claude. It can describe your website goals, design rules, content style, coding preferences, and what Claude should avoid.

This is one of the most useful parts of the Claude Code for WordPress workflow. Instead of repeating the same instructions every time, you document the rules once and let Claude use them as project context.

### Step 3: Train Claude Around a Web Design Style

Timestamp: 00:03

Claude becomes more useful when it understands your design style. In the video, the workflow includes training Claude around practical web design preferences: strong section hierarchy, clean spacing, premium landing page patterns, clear CTAs, and layouts that feel intentional.

This helps prevent generic AI output. If you want high-converting Elementor design with AI, you need to guide the design direction instead of asking for "a nice page" and hoping the result works.

### Step 4: Build Elementor AI Skills

Timestamp: 00:04:20

The tutorial then shows the idea of building Elementor-focused AI skills. These skills can help Claude think about Elementor containers, section structure, widget decisions, responsive behavior, and landing page flow.

This is different from a normal prompt. A skill can hold a consistent role, such as Elementor designer, WordPress architect, or landing page reviewer. That makes the AI easier to use across multiple website builds.

### Step 5: Review the Final Claude AI Website Outcome

Timestamp: 00:05:47

After setting up the workflow, the video reviews the final website outcome. This step matters because AI-generated design still needs human review.

Look at the spacing, mobile layout, visual hierarchy, typography, CTA placement, and whether each section supports the page goal. AI can speed up the process, but the designer still needs to judge whether the final page works for real visitors.

### Step 6: Understand MCP and Claude Architecture

Timestamp: 00:06:32

MCP helps connect Claude to external tools and systems. In this workflow, the architecture matters because Claude is not only answering questions. It is being connected to WordPress in a more direct and structured way.

Understanding the architecture helps you avoid treating the workflow like magic. You need to know what Claude can access, what it can change, and what still requires careful approval and testing.

### Step 7: Connect Claude AI to WordPress with Novamira

Timestamp: 00:07:06

The Novamira plugin is used to help connect Claude AI to WordPress through the MCP workflow. This setup gives Claude a bridge into the WordPress environment so it can support more practical website work.

For beginners, the key point is to move slowly. Connecting AI to WordPress can be powerful, but you should understand the plugin setup, permissions, and the type of changes Claude is allowed to make.

### Step 8: Configure MCP Credentials and JSON Settings

Timestamp: 00:08:26

The tutorial includes editing MCP credentials and JSON configuration. This is one of the technical parts of the workflow, and it should be done carefully.

Credentials and JSON settings tell Claude Code how to connect to the right system. A small mistake can break the connection or point the workflow to the wrong place. Always double-check the formatting, site details, and credentials before moving forward.

### Step 9: Use Claude Code to Edit WordPress and PHP

Timestamp: 00:09:48

Claude Code can help edit WordPress-related files and PHP functions, but this is also where beginners need to be careful. PHP changes can affect the behavior of the website, so every edit should be reviewed and tested.

Use Claude to explain the function, suggest the change, and help debug issues. Do not blindly accept PHP edits without understanding what the function does and how it affects the site.

### Step 10: Connect Elementor with the Claude AI Workflow

Timestamp: 00:10:36

The final workflow connects Elementor design decisions with Claude AI support. This is where the setup becomes useful for landing page creation: Claude can help plan sections, improve content flow, review layout structure, and support a more consistent design process.

The best results happen when you separate responsibilities. Let Claude help with structure, design review, and implementation support, while you keep control over the final creative and business decisions.
